#算法

递归取左子树和右子树的最大值即可，每次递归结束深度 +1：

```go
func maxDepth(root *TreeNode) int {
	if root == nil {
		return 0
	}
	return max(maxDepth(root.Left), maxDepth(root.Right)) + 1
}
```

java:

```java
public int maxDepth(TreeNode root) {
	if (root == null) {
		return 0;
	}
	return Math.max(maxDepth(root.left), 
					maxDepth(root.right)) + 1;
}
```
