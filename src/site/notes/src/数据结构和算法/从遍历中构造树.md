#算法 #树 

此类问题，需要掌握各种遍历结果数组的特点。

前序遍历：

```java
[ 根节点, (左子树), (右子树) ]
```

中序遍历：

```java
[ (左子树), 根节点, (右子树) ]
```

后序遍历：

```java
[ (左子树), (右子树), 根节点 ]
```

做法（**递归 + 分治**）：
1. 先从前序（第一个）或者后序（最后一个）找到**根结点**
2. 再到中序遍历中算出**左子树个数**，即：`rootIndex - leftIndex`
3. **递归**进行构造
