#算法 

## 概念

回溯（*Backtrack*）是深度优先遍历的一个特有现象。

![](/img/user/attachments/images/1611540618-qrHmGi-image.png)

> [!INFO] 
> 回溯算法和 DFS 算法的细微差别是：
> - 回溯算法是在遍历「**树枝**」
> - DFS 算法是在遍历「**节点**」

## 模板

```python
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```
