基于 ExGraph 的中间件 MIDWARE 机制，设计 CommonDealPhpOrGoStrategy 算子：
- migrate_switch_name：以类名 + 方法名组成，对应 PHP 策略的类名和方法名
- migrate_switch_sample：抽样实验的实验变量
- migrate_switch_value：是否开启


## PHP

