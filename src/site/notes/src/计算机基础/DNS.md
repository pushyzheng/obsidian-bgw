#计算机网络 #HTTP #应用层协议 

## DNS 使用的什么传输层协议？

DNS 既使用了 UDP，也使用了 TCP：

1. **区域传送**（主域名服务器向辅助域名服务器**传送变化数据**）使用的是 TCP
2. **查询域名**（域名解析）时是 UDP

## DNS 域名解析都有哪几种方式？

包含迭代和递归查询，其区别在于，当客户端向 DNS 服务器请求，若无法解析：

- ==迭代查询==
	- 返回更接近目标的 DNS 服务器地址
	- 客户端**逐级**查询，直到得到结果
- ==递归查询==
	- 会**代替**客户端向其他 DNS 服务器请求
	- 直到找到结果并返回，保证客户端获得完整结果
